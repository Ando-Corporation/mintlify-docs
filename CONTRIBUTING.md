# Contribute to the Ando docs

## Local development

Run the Mintlify preview from this directory:

```bash
mint dev
```

## Writing guidelines

- Use direct, active voice.
- Address the reader as `you`.
- Keep headings in sentence case.
- Prefer concrete examples over generic descriptions.
- Use Mintlify components when they make the page easier to scan.
- Run `mint validate` and `mint broken-links` before publishing docs changes.
