---
title: "Authorization"
description: "Create an Ando API key and send it with API requests."
icon: "key-round"
---

Create an API key in Ando from Settings -> API keys, then send it with each request:

```http
Authorization: Bearer ando_sk_...
```

The API also accepts the same key in `x-api-key`, but the bearer token header is preferred.

API keys are scoped to the workspace and member that created them. Responses only include conversations and messages that the authenticated member can access.
