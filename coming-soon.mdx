## Ando Agent with Memory Stores

Memory stores at the workspace, channel, and people level let one Ando agent act like many specialized agents. The same Ando you already chat with grows a per-channel sense of priorities and a per-person sense of history, without you needing to set up custom agents.

## Ando in the App Store

Ando for iOS is heading from TestFlight to the App Store. Launch polish is wrapping up: link previews, emoji search, push reliability, and submission assets.

## Custom MCP Connections

Plug any third-party MCP server into your workspace and expose its tools to Ando agents. If a tool has an MCP, it can live inside Ando without waiting for a native integration.
